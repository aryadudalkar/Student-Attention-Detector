\# Smart Classroom Attention Detection



This project uses YOLOv8 to detect whether students are attentive or distracted.



\## Installation



1\. Clone the repository



git clone https://github.com/YOUR\_USERNAME/smart-classroom-attention.git



cd smart-classroom-attention



2\. Install dependencies



pip install -r requirements.txt



\## Run Webcam Demo



python scripts/webcam\_test.py



Press ESC to exit.
